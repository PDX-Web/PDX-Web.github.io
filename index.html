Class Description and Goals

This class will introduce various web techniques and cover the spectrum of skills required to become a full-stack developer. You will not become an expert in ten weeks, but you will leave this course with a better understanding of what skills you may want to continue improving and what areas of web development you may wish to pursue.

In this course, you will learn how to create web applications using a combination of front-end and back-end technologies, you will build both a personal website or a full-stack application. Additionally, you will gain a lot of experience reading documentation, using a code editor, and using version control to submit all the assignments in this class.
